# SkillSense

This repository contains the SkillSense AI app.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Create a `.env` file with your Gemini API key:
   `GEMINI_API_KEY=<your-api-key>`
3. Run the app:
   `npm run dev`

## Deployment

Follow standard Vite deployment steps for your chosen hosting provider.
